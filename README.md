# Student Management System

A robust backend application developed using **Java** and **Spring Boot REST API** to streamline student record management. Designed with clean layered architecture (**Controller, Service, Repository**) for scalability and maintainability. Implements complete CRUD operations to efficiently add, view, update, and delete student details. This project demonstrates strong skills in backend development, RESTful API design, and Java application structure.

### Tech Stack

* Java
* Spring Boot
* REST API

### Key Features

* Create Student Records
* Retrieve All / Individual Student Data
* Update Student Information
* Delete Student Records
* Structured and Maintainable Code Design
